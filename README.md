# WeatherApp
Tabs done in SCSS; Content with lazy loading.

# Installation guide

## Easy way

   This way requires you to have **NodeJS**, **Git Bash**, **BowerJS** and **GULP** installed.
   - Clone repository.
   - CD into the project.
   - Run *npm intall* in root of the project.
   - Run *bower intall* in root of the project.
   - Run *gulp webserver*
